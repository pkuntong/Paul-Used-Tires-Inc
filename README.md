# Paul's Used Tires & Rims

Welcome to the codebase for **Paul's Used Tires & Rims**, located in Gaithersburg, MD.

## About
Quality used tires and rims at unbeatable prices. Trusted by the Gaithersburg community for reliable service and expert tire installation.

## Services
- **Used Tires**: Wide selection of quality inspected used tires for all vehicle makes and models at affordable prices.
- **Tire Installation**: Professional mounting, balancing, and installation by experienced technicians.
- **Rims & Wheels**: Find the perfect rims to match your style. New and used options available.
- **Wheel Repair**: Expert wheel and rim repair services to get you back on the road safely.

## Technologies
This project is built with:
- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

To run this project locally, ensure you have Node.js installed, then follow these steps:

```bash
# Install the necessary dependencies
npm install

# Start the development server
npm run dev
```
